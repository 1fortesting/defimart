# **App Name**: DEFIMART

## Core Features:

- User Authentication & Profile Management: Secure user registration and login via email/password, along with management of user profiles (display_name, avatar_url, phone_number) stored in Supabase.
- Product Listing & Browsing: Sellers can create, view, edit, and delete their product listings including name, description, price, and multiple image uploads. Buyers can efficiently browse available products.
- Real-time Buyer-Seller Chat: Facilitate direct, real-time messaging between buyers and sellers regarding specific products, utilizing Supabase Realtime to display unread message counts.
- Pickup Order Management: Buyers can place pickup-only orders for products. Sellers manage order details and update their status (pending, ready, completed) which triggers in-app notifications for buyers.
- AI Product Description Assistant: A generative AI tool to assist sellers in crafting concise and appealing product descriptions based on input keywords and essential product details.

## Style Guidelines:

- Primary color: A vibrant digital blue (#2680E6) to convey trust, clarity, and innovation. (HSL: H 220, S 70%, L 50%)
- Background color: A very light, subtle grey-blue (#EBF0F6) for a clean, expansive interface that promotes readability. (HSL: H 220, S 15%, L 95%)
- Accent color: A bright, energetic cyan (#33CCFF) to highlight interactive elements and key information. (HSL: H 190, S 80%, L 60%)
- Headlines and Body text: 'Inter', a modern sans-serif typeface, chosen for its excellent readability and neutral, objective aesthetic, suitable for technical content and clear communication.
- Employ a set of clean, line-based vector icons for actions and categories to maintain a modern and professional visual language throughout the application.
- Implement a responsive, grid-based layout for product listings and user dashboards, prioritizing content readability and intuitive navigation across all device sizes.
- Utilize subtle, smooth animations for state changes, component transitions, and interactive feedback, enhancing user experience without being distracting.
